# 📊 Marketing Campaign Analysis (EDA)

## 📌 Project Overview
This project focuses on performing Exploratory Data Analysis (EDA) on a marketing campaign dataset to uncover key business insights. The goal is to analyze customer behavior, campaign performance, and identify which marketing channels provide better returns.

---

## 🎯 Objective
- Clean and analyze the dataset  
- Understand customer demographics and spending patterns  
- Evaluate campaign performance  
- Identify high-performing marketing channels  
- Provide recommendations based on insights  

---

## 🛠️ Tools & Technologies
- Python  
- Pandas  
- Matplotlib  
- Seaborn  
- VS Code  

---

## 📂 Dataset
- File Name: `marketing_campaign.csv`  
- Contains customer details such as:
  - Age (Year_Birth)  
  - Income  
  - Spending on products  
  - Campaign responses  
  - Purchase channels  

---

## 🔧 Data Cleaning
- Removed duplicate records  
- Handled missing values  
- Standardized column names  
- Created new features:
  - Age  
  - Total Spending  

---

## 📊 Exploratory Data Analysis
The following analyses were performed:
- Age distribution of customers  
- Income vs Spending relationship  
- Campaign response analysis  
- Channel-wise purchase behavior  

---

## 📈 Key Insights
- Customers with higher income tend to spend more  
- Web purchases contribute significantly to total sales  
- Campaign response rate is relatively low  
- Certain channels perform better than others  

---

## 📉 Funnel Analysis
- Total Customers  
- Customers Responded to Campaign  

This helps understand the conversion rate of marketing efforts.

---

## 💡 Recommendations
- Focus more on high-performing channels (e.g., web marketing)  
- Target high-income customer segments  
- Improve campaign strategies to increase response rate  
- Optimize budget allocation based on ROI
